# ファイルオブジェクト (struct file)

宣言: `linux/include/linux/fs.h:1259`

参考：詳解Linuxカーネルp500


Linuxプログラミングインタフェースp102図5-2のオープンファイルテーブルに相当する情報を持つ（ただし、i-nodeポインタは持っていない）。 
メンバ例：
- `f_op`: ファイル操作テーブルへのポインタ。ファイルシステムごとに固有のファイル操作が定義されている
- `f_pos`: 現在のファイルオフセット
- `f_flags`: ファイルのオープン時に指定されたフラグ
